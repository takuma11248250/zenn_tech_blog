# 最近よく感じること

全員が会社のVMV（ビジョン・ミッション・バリュー）達成のために視座高く仕事しているわけではない、ということをひしひし感じています。

同じ組織に属しているからといって、同じ目的や思考を持っているとは限りません。当たり前のことのように聞こえるかもしれませんが、これを身をもって体感したとき、じわじわ効いてきます。

# 自分の仕事スタンス

プロジェクトを進めるとき、自分はこういう順番で考えます。

まず「何のためにやるのか」「会社にどんなメリット・デメリットがあるか」「誰を巻き込む必要があるか」「何をやって、何をやらないと決めるか」——考えられるあらゆる事象を全部テーブルに並べ、グルーピングして整理した上でロードマップを作ります。

ロードマップができたら5W1Hで肉付けし、資料は2種類に分けて作ります。

- **経営陣向け**：必要なことをシンプルに。質問が来たら答えるパターン
- **現場向け**：なぜやるのか、どういう時系列で、誰に何を求めているか、の具体の話

この2種類で全員の認識を合わせてから「ここからは各専門家に任せる」というスタイルで動いています。全部を細かくやる必要はないですが、少なくとも「ゴールの認識」と「今何をやるべきか」は常に合わせてきました。

ここで特に意識しているのは、**HOWは一番最後**だということです。どうやってやるか（How）の話は、ゴールや目的が明確になってから初めて意味を持ちます。ところが実際の現場では、根本の目的を考えていない、あるいはそもそも知らないままHOWの話が先行することがよく発生します。「これどうやって実装する？」「このツールを使えばできる！」という会話が、なぜそれをやるのかを整理しないまま走り出してしまうケースは、本当によく見てきました。

# 熱量の違いが組織を変える

自分はベンチャーマインドが強いほうで、これまでそういう人たちと一緒に仕事してきました。会社の夢や目標に向かって熱く議論するし、仕事の進め方やスタンスもほぼ全員が同じ志を持っていることが多かったです。

でも、このマインドは**会社が大きくなればなるほどずれていきます**。人によって仕事への熱量が全然違うし、会社のVMVに共感して「この業界をこう変えたい」と思っているわけでもない人が増えていく。

熱が同じ人とは、たくさん議論できます。「もっとこれやってこう」「これも面白そう」「いや、これはまだ手をつけないほうがいい」「そもそもこれって今いらないよね？」——目線が同じだからコミュニケーションも仕事の任せ方も、お互いに信頼して委ねられる状態になれます。

でも、熱い人が少数で、「今はこれでいいんじゃない？」「なんでそんなに熱く議論するんだろう」と冷めた状態の人が多数派になると、何が起きるか。

**頑張ってよくしようとしている人が、孤立します。**

「自分たちだけが無駄に頑張ろうとしてる」「この会社でこんなに頑張る必要はあるのだろうか」——そう思って、超優秀な人たちが離れていくことも多いと思います。優秀な人たちは優秀な人たち同士で集まりますし、逆も然りです。

# 強い組織とは

ただここで言いたいのは、「全員が視座を高く持て」ということではありません。

自分の強みを最大限発揮できる環境が整っていて、「自分の役割はこれだから、この領域では誰にも負けないパフォーマンスを出す」——そういう人たちが集まっている状態が、一番強い組織だと思っています。

わかりやすい例で言えば、ワンピースのイメージに近いです。それぞれが自分の夢を持ちながら、同じ夢（船長を海賊王にする）も共有している。船長、剣士、コック、航海士、医師——それぞれが専門領域で最大限のパフォーマンスを発揮し、お互いに助け合い、背中を信頼して任せられる状態。これが組織の目指すべき姿だと思っています。

# だから採用めっちゃ大事

採用が組織のすべてを決めると言っても過言じゃないと思います。

「採用頑張ろう」は誰でも言えます。でも実際にカルチャーフィットを言語化して、それを採用基準に落とし込んで、ちゃんと見極めている会社は少ないです。スキルや経験は後から伸ばせますが、仕事への向き合い方や熱量はそう簡単には変わりません。

採用で見極めるべきは、技術力や経験といった表面的な部分だけではありません。**会社のVMV（ゴールや夢）にどれくらい共感しているか、どれくらいの熱量を持っているか、会社が求める役割とその人が提供できる役割が100%マッチしているか**——この3点を採用で見極めることが最優先事項だと思っています。

# まとめ

仕事はWHYから始まり、HOWは一番最後。そしてそのWHYを共有できる人、自分の専門領域で誰にも負けないパフォーマンスを出せる人を採用で集められるかどうかが、組織の熱量と強さを決めます。

「採用大事」を本当に実践できている会社はまだまだ少ないと思います。だからこそ、採用にもっと真剣に向き合い、仕組み化し続けることが、これからの組織づくりの鍵になると強く感じています。
