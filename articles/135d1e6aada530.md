## 優秀なデータエンジニアに必要な5つの要素

私が考える要素は以下の5つです。

:::message
1. 技術力
2. コミュニケーション能力
3. 主体性・自立性
4. 問題解決力
5. チーム貢献度
:::

この5つを定量的に評価できるようにして、それぞれの軸を上げるために何をするかを自分で決め、行動に落とし込んで仕組み化していく。そういうサイクルを回せる人が増えれば、優秀なデータエンジニアは自然と育っていくと思っています。

## 1. 技術力

**技術力は必須です。ただし、これだけでは意味がありません。**

高度な技術知識とスキルを持ち、設計やアーキテクチャにも主導的に関与できる。チーム内の技術的支柱になっているレベルであれば最高ランクと言えます。

ただ、コミュニケーションが全くできず、自分の言うことが全て正しいと思っている人だったらどうでしょうか。そういう人についていきたいとは思いませんし、一緒に何かを作ろうとも思いません。他の部署とも常に摩擦が起き、プロジェクトが前に進まない状態になります。

技術力は「持っていて当たり前」のベースラインです。それだけを磨き続けているだけでは、チームの中での価値は頭打ちになります。

## 2. コミュニケーション能力

**個人的には、5つの中で一番重要だと思っています。**

状況に応じて適切な伝達ができ、チーム内外で信頼されている状態。対話や議論の質も高い状態が最高ランクです。

一人で仕事をするのであれば不要かもしれません。しかし、会社に所属している以上、それはあり得ません。必ず同僚がいて、上司がいます。

コミュニケーションが取れないと、こういうことが起きます。

- 報告・連絡・相談がない
- 認識がずれたまま作業が進む
- 些細なことでハレーションを起こす
- すぐ不機嫌になる
- 「正しく評価されていない」と文句を言う
- 愚痴ばかりで行動しない

心当たりのある方は周りにいるのではないでしょうか。**コミュニケーション能力は、スキルというより姿勢の問題でもあります**。どれだけ技術が高くても、周囲との連携が取れなければ、その力は半分以下にしか発揮されません。

## 3. 主体性・自立性

次のステップに進むためには、**自分で仕事を見つけ、能動的に動いていく**必要があります。

最初は上司に依頼されたことをこなして評価される状態でも構いません。ただ、それがいつまでも続くとは思わないでください。上司がずっといてくれるとは限りませんし、仕事を供給し続けてくれるとも限りません。

受け身のまま仕事を待っている状態では、近い将来、AIに仕事を取られます。これは脅しではなく、すでに起きていることです。

**自分から問題を見つけ、解決策を提案し、実行するまでを自分でドライブできる人**が、これからの時代に生き残るデータエンジニアだと思っています。

## 4. 問題解決力

問題解決力は、言い換えれば**諦めない力**でもあります。

仕事を進めていると、面倒なことややりたくないことの方が圧倒的に多いです。そこですぐに投げ出して人任せにしてしまう人は、残念ながら一生成長しないと思っています。

誰よりも困難な課題に立ち向かい、最後まで諦めず、冷静に状況を分析し、その時々で最適な解決策を導き出せる。そういう人が周囲から信頼され、頼られる存在になります。

データ領域の仕事は、特にこの力が問われます。データの品質問題、パイプラインの障害、要件の曖昧さ、ステークホルダーとの調整など、一筋縄ではいかない問題が日常的に発生します。**そこで粘れるかどうかが、エンジニアとしての差になります。**

## 5. チーム貢献度

会社に所属している以上、必ずどこかのチームに入って仕事をしていると思います。

「自分さえよければいい」という姿勢で仕事をしている人は一定数います。私はそういう人と一緒に仕事をしたいとは思いません。チームのために積極的に支援・共有・フォローを行い、文化的にもいい影響を与えられている人と仕事をしたいと思っています。

ここで誤解がないよう補足すると、「会社のため」「チームのため」と自分を犠牲にすることを求めているわけではありません。人間である以上、自分の人生があり、守るべきものがあります。自分が大切なのは当然のことです。

ただ、**仕事の向かい先を「自分のため」ではなく「何かを成し遂げるためにみんなで」という方向に向けてほしい**ということです。

会社と社員はwinwinの関係であるべきだと思っています。何かを成し遂げたいという目標があり、それを実現するために会社という環境を使わせてもらう。その会社には同じゴールを持った人が集まっている。だからみんなで同じ方向に向かっていく。これが理想の姿です。

もちろん、金銭のために仕事をしているという人がほとんどでしょう。それは否定しません。ただ、人生のうち8割近くを仕事に使うのに、「お金のためだけ」という動機だけで過ごすのは、非常にもったいないと思っています。

**何かを成し遂げるためにみんなで、という感覚を持って仕事をしている人と、一緒に働きたいと思っています。**

## まとめ

データエンジニアに必要な要素を5つ挙げました。

| 要素 | 一言で |
|---|---|
| 技術力 | ベースラインとして必須。ただしこれだけでは不十分 |
| コミュニケーション能力 | 最も重要。技術力を最大化するための土台 |
| 主体性・自立性 | 受け身はAIに置き換えられる |
| 問題解決力 | 諦めないで粘れるかどうか |
| チーム貢献度 | 「みんなで成し遂げる」という方向に向かえるか |

これらを定量的に評価し、自分の弱い軸を把握して、何をするかを決めて行動に落とし込む。そのサイクルを自分で回せるようになれば、成長のスピードは格段に上がります。

技術力は必要条件です。ただし、**それだけを磨き続けても、チームの中での価値は頭打ちになります。**残りの4つの軸も意識して、バランスよく成長していくことが、優秀なデータエンジニアへの近道だと考えています。
